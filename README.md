# Career Copilot

## AI-powered local-first career assistant built with Python and Mistral.

### Career Copilot helps automate and optimize the entire job search workflow:

Resume intelligence & parsing
Structured profile extraction
Job discovery & ranking
Resume tailoring
Cover letter generation
Application automation
Features
Local LLM support via Mistral + Ollama
Resume parsing into structured JSON
Resume knowledge/profile generation
Job scraping & normalization
Semantic job matching
ATS resume optimization
Cover letter generation
Browser automation for applications
Modular domain-driven architecture
SOLID-compliant codebase
Typed Python + Pydantic models
Structured logging
Environment-based configuration
Tech Stack
Component	Technology
Backend	FastAPI
LLM	Mistral (Ollama)
Embeddings	SentenceTransformers
Vector Store	FAISS
Database	MongoDB
Automation	Playwright
Parsing	PyMuPDF
Config	Pydantic Settings
